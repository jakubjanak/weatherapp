Jednostránková webová aplikace, která se odkazuje na OpenWeather API. Tuto aplikaci vytvářím, protože mi nevyhovuje žádná jiná webová aplikace. Chci jednoduchou webovou aplikaci, kde uvidím aktuální data v jednoduchém designu a jednoduchou předpověď na týden dopředu.

Projekt je založený pomocí Vite a tvořený stackem React + Typescript + Tailwind.
